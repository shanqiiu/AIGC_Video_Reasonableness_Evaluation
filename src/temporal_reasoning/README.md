# 时序合理性分析模块

## 模块概述

时序合理性分析模块是AI生成视频合理性评估系统的核心模块之一，主要负责检测视频中运动与结构的时序连贯性。

**核心功能**：
- 检测运动与结构的时序连贯性
- 识别违反物理规律和人体结构的时序不一致性问题
- 输出运动合理性得分、结构稳定性得分和异常实例列表

## 技术组件

本模块整合以下技术组件：

1. **光流分析**：采用RAFT等先进光流算法评估全局运动平滑度
2. **实例分割与跨帧追踪**：集成Grounding DINO + SAM + DeAOT等技术，实现文本引导的实例分割与稳定跨帧追踪
3. **关键点/动作自然性分析**：基于MediaPipe等框架提取人体关键点，分析眨眼、嘴型、手势等生理动作的自然性

## 输入输出

**输入**：
- 视频帧序列（RGB图像）
- 可选：文本描述（用于引导检测特定部位）

**输出**：
- 运动合理性得分（0-1）
- 结构稳定性得分（0-1）
- 异常实例列表（包含异常类型、时间戳、置信度等）

## 典型应用场景

- **结构消失异常**：小孩吐舌头过程中舌头突然消失、手指在运动中突然断裂
- **运动突变异常**：物体运动速度突然改变、运动方向非连续变化
- **生理动作异常**：眨眼频率异常、嘴型变化不自然、手势动作不连贯
